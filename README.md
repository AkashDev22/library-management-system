# Library Management System

## Overview
This project is a Java-based console application that manages library operations using the Java Collections Framework.

## Features 
1. Add Book
2. Remove Book
3. Register Member
4. Issue Book
5. Return Book
6. Search Books
7. List Available Books
8. List All Books
9. List Members

## Technologies Used
- Java
- Java Collections Framework
- OOP Concepts

## How to Run
1. Open the project in IntelliJ IDEA
2. Run the LibraryManagementSystem Java file
